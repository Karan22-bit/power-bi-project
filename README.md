# Data Jobs Dashboard w/ Power BI

![Dashboard Image](/IMAGESS/IMAGE1.png)

## Introduction

This dashboard is created for **Job seeker, Job transitioners, and Job swappers** to solve a common problem: information about the data jobs market is scattered and hard to grasp.using a real world dataset of 2024 data science job postings (including titles,salaries and location), this project provides a single, easy to use interface to explore market trends and composition.

## Skill Showcased

- **Data Transformation(ETL) with Power Query**: Cleaned,shaped, and prepared the raw data for analysis by handling blanks, changing data types, and creating new columns.
- **Implicit Measures:** Formulated measures to drive key insights and KPIs like **Median yearly Salary** and **Job Count**
- **Core Charts:** Utilized **column,bar,line** and **area charts** to compare job counts and track trends over time.
- **KPI Indicators & tables:** Used **Cards** to display key metrics and **Tables** to provide granular sorted data
- **Dashboard design:** Designed and intuitive and visually appealing layout,exploring both Common and Un common chart types to best tell the story.
- **Interactive Reporting:**
  - **Slicers:** To Dynamically filter the report by jo title
  - **Buttons and Bookmarks:** To create a seamless navigation
